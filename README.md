# Week 1 Semantic Website

## 📌 Project Overview

This project is a personal portfolio website created for our Week 1 Semantic HTML activity. The website demonstrates the proper use of HTML5 semantic elements while following basic web accessibility practices.

The portfolio consists of three pages:

- **Home** – Introduces the designer, showcases projects, and displays software skills.
- **About Me** – Shares personal background, education, and hobbies.
- **Contact** – Provides a contact form and contact information.

## 📄 Semantic HTML Elements Used

This project uses HTML5 semantic elements to improve readability, organization, SEO, and accessibility.

| Element | Purpose |
|---------|---------|
| `<header>` | Displays the page title and navigation menu. |
| `<nav>` | Contains links to navigate between pages. |
| `<main>` | Holds the primary content of each page. |
| `<section>` | Groups related content into meaningful sections. |
| `<article>` | Contains independent content on the About page. |
| `<table>` | Displays software skills in a structured format. |
| `<form>` | Collects user contact information. |
| `<footer>` | Displays copyright information. |

---

## ♿ Accessibility Features

This website includes several accessibility improvements:

- Semantic HTML5 structure
- Proper heading hierarchy (`h1`, `h2`, `h3`)
- Descriptive `alt` text for all images
- Labels connected to form inputs
- Table headers using `<th>` and `scope`
- Responsive viewport meta tag
- Video controls for media playback
- Keyboard-accessible navigation
